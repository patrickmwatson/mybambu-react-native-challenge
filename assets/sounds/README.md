# Sound Assets

This directory contains sound files for the MyBambu app.

## Click Sound

For the click sound, you can download a free click sound from:
- https://freesound.org/search/?q=click+button
- https://www.zapsplat.com/sound-effect-category/button-clicks/

Recommended: A short (< 0.5 second) subtle click or tap sound.

Save it as `click.mp3` in this directory.
